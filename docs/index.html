A NumPy-compatible numerical computing library
for modern C++

libnpc++ is a C++17 RAII wrapper around libnpc — ~335 functions across 21 modules. Zero Python dependency. Header-only. MIT licensed.

C++17Header-onlyCMakemacOS & Linux
Get Started → GitHub

RAII Ownership

Automatic memory management — arrays free themselves. No manual free_array() calls needed.

Operator Overloading

a + b, a * 2.0, -a, 2.0 / a, a += b — all element-wise with broadcasting.

Natural Indexing

a(0, 1), a[3], a.at<double>({i,j}), a.scalar<T>() — typed access everywhere.

Stream Output

std::cout << a prints metadata. a.print() dumps values.

Move Semantics

Efficient ownership transfer. Copy is deep; move is zero-cost.

Linear Algebra

SVD, QR, LU, Cholesky, Eig, matrix inverse — all with RAII result structs.

Quick Start

A 30-line tour of the library:

main.cpp
#include <npc/npc.hpp>

int main() {
    // ── Create arrays ─────────────────────────
    auto a = npc::ones({3, 3});           // 3×3 of 1.0
    auto b = npc::full({3, 3}, 2.0);      // 3×3 of 2.0
    auto r = npc::arange(0.0, 10.0, 2.0); // [0,2,4,6,8]
    auto I = npc::eye(3);                 // identity

    // ── Operators ─────────────────────────────
    auto c = a + b;        // element-wise add
    auto d = c * 3.0;      // scalar multiply
    auto e = 2.0 / a;      // scalar division
    e += 1.0;              // compound assignment

    // ── Indexing ──────────────────────────────
    std::cout << a(0, 1) << std::endl;  // → 1.0
    a[0] = 99.0;           // bracket access (1-D)
    double v = a.at<double>({1, 2});

    // ── Math & stats ─────────────────────────
    auto s = npc::sqrt(b);
    auto m = npc::mean(r);
    auto t = npc::sum(a, 0);  // along axis 0

    // ── Linear algebra ────────────────────────
    auto svd_r = npc::svd(I);  // → {U, s, Vt}
    auto inv = npc::inv(a);

    // ── Cleanup ───────────────────────────────
    return 0;  // RAII: all arrays freed automatically
}

Array Class

The npc::Array class is the central type of libnpc++. It wraps a raw libnpc ::Array* with RAII semantics via std::unique_ptr, providing automatic memory management, operator overloading, typed element access, and C++17 copy/move semantics.

Constructors

ConstructorDescription
Array()Default: creates a null array wrapping nullptr. Use is_null() or operator bool() to test.
Array(::Array* ptr, bool owned=true)Takes ownership of a raw C ::Array*. If owned=false, deep-copies instead.
Array(const Array& other)Deep copy constructor. The new array is completely independent of the original.
Array(Array&& other) noexceptMove constructor. Transfers ownership — the source becomes a null array. Zero-cost.

Assignment

OperatorDescription
Array& operator=(const Array&)Deep copy assignment. Source unchanged, target becomes independent copy.
Array& operator=(Array&&) noexceptMove assignment. Transfers ownership from source to target.

Accessors

MethodReturnDescription
ndim()intNumber of dimensions. 0 for scalar, 1 for vector, 2 for matrix, etc. Returns 0 for null arrays.
size()intTotal number of elements (product of all shape dimensions).
shape()const int*Pointer to shape array of length ndim(). Returns nullptr for 0-d or null arrays.
shape_vec()vector<int>Shape as a convenient std::vector<int>.
strides()const int*Pointer to strides array. May be nullptr.
dtype()DTypeElement data type as a scoped npc::DType enum.
is_view()boolTrue if this array is a view sharing another array's data buffer.
is_null()boolTrue if the array wraps no data (nullptr).
operator bool()boolExplicit conversion: true if non-null. Enables if (arr) { ... }.

Element Access

Typed Access: at<T>({i, j, ...})

Returns a reference (T&) to the element at given multi-dimensional indices. The number of indices must match ndim() (assert-checked).

double& v = a.at<double>({1, 2}); // 2-D: row 1, col 2 int64_t& iv = a.at<int64_t>({0}); // typed Int64 access

Variadic operator()(i, j, k)

Variadic-template multi-dimensional indexing returning double. Accepts any number of int indices.

double v = a(0, 1); // 2-D access double w = a(2); // 1-D access

Bracket operator[](int)

Single-index access for 1-D arrays. Returns double& reference. Requires ndim()==1.

double v = a[3]; // read a[0] = 99.0; // write

scalar<T>()

Returns the first element as type T. Ideal for extracting 0-d scalar results from aggregations.

auto total = npc::sum(data); double v = total.scalar<double>(); // extract scalar

Ownership Semantics

OperationBehaviorCost
Copy constructDeep copy — allocates new buffer, copies all elementsO(n)
Copy assignDeep copy — frees old data, allocates new, copies elementsO(n)
Move constructTransfers internal pointer — source becomes nullO(1)
Move assignTransfers internal pointer — source becomes nullO(1)
DestructorCalls free_array() on the underlying C arrayO(1)
RAII guarantee: When an npc::Array goes out of scope, its destructor automatically frees the underlying libnpc C array. You never need to call free_array() manually. This is the fundamental safety improvement over the raw C API.
